# 🚢 Titanic Data Preprocessing & Cleaning Project

## 📌 Overview
This project focuses on **data collection, preprocessing, and cleaning** using the famous Titanic dataset.  
The goal is to transform raw data into a clean and structured format suitable for Machine Learning models.

This task was completed as part of my **Machine Learning Internship at Spark Que Tech Solutions**.

---

## 🎯 Objectives
- Understand the importance of data preprocessing
- Clean and prepare raw data for ML models
- Apply real-world data handling techniques

---

## 🛠️ Technologies Used
- Python 🐍
- Pandas
- NumPy
- Matplotlib / Seaborn

---

## 📂 Project Workflow

### 1️⃣ Data Collection
- Loaded Titanic dataset using Pandas

### 2️⃣ Data Cleaning
- Handled missing values
- Removed duplicate records
- Fixed incorrect data types

### 3️⃣ Data Preprocessing
- Encoded categorical variables
- Normalized numerical features

### 4️⃣ Outlier Detection
- Used boxplots to detect outliers

### 5️⃣ Final Output
- Saved cleaned dataset for further Machine Learning use

---

## 📊 Key Learnings
- Importance of clean data in ML
- Handling missing and inconsistent data
- Feature transformation techniques
- Data visualization for outlier detection
